📥 Powerful Download Manager

Download files via HTTP, HTTPS, and FTP with resume support and up to 10 parallel downloads. Auto-retry on failure and smart error handling ensure your downloads complete successfully.

  • Resume broken downloads
  • Up to 10 simultaneous downloads
  • Speed limit control (KB/s - MB/s)
  • Auto-retry on network errors (up to 5 times)
  • FTP support with authentication
  • Download history with statistics

🌐 Built-in Web Browser

Browse the internet with multiple tabs, manage bookmarks with folders and drag-and-drop, and enjoy automatic ad blocking. Import/export your bookmarks from Chrome or Firefox.

  • Multiple tabs with tab management
  • Bookmark folders & drag-drop organization
  • Ad blocker with custom filters
  • Import/Export bookmarks (Chrome/Firefox/HTML)
  • Dark/Light themes sync with app
  • Private browsing mode

🧲 Torrent Download Manager

Download torrent files and magnet links with ease. Select which files to download before starting, and monitor seeds/peers in real-time with detailed graphs.

  • .torrent & magnet links support
  • File selection before download
  • Seed/peer monitoring with real-time graphs
  • Pause/Resume torrents with state save
  • Sequential downloading mode
  • DHT, PEX, and LSD support

🎬 YouTube Downloader

Download videos from YouTube in various qualities (up to 1080p/4K) or extract audio as MP3. Full playlist support and smart browser integration for one-click downloads.

  • Video download (up to 1080p, 4K support available)
  • MP3 extraction (128k/192k/320k bitrate)
  • Playlist support (entire playlists)
  • Thumbnail preview before download
  • Smart browser button for one-click detection
  • Subtitle download (SRT, VTT formats)

📅 Smart Scheduler

Schedule downloads for specific times, create recurring tasks (daily, weekly), and organize them into queues. Perfect for off-peak hours to save bandwidth.

  • One-time or recurring tasks (daily, weekly, monthly)
  • Multiple download queues with priorities
  • Start/Stop time configuration
  • Post-download actions (shutdown, sleep, notify)
  • Calendar view of scheduled tasks
  • Email notification on completion

🌍 Bilingual Interface

Fully supports both Arabic (RTL) and English. Switch between languages instantly, with complete RTL layout support for Arabic users.

  • Full Arabic translation (100% complete)
  • RTL layout support with proper alignment
  • One-click language switch

  • Right-to-left support for all UI elements

📋 Clipboard Monitor

Automatically detects copied URLs and shows a quick-add dialog. Choose download options, rename files, and select save path instantly.

  • Auto URL detection (HTTP, HTTPS, FTP, magnet links)
  • Quick-add dialog with URL preview
  • File rename option before download
  • Custom save path selection per download
  • Ignore list for specific sites
  • Sound notification on detection
